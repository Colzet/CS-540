This project is for CS 540 DataBase Management System. Our project is to make a website to provide an average score of a movie in 3 movie review page--IMDB, web 2, and web 3. Our website is just a simple window to show the result. The key point of this project is to integrate datasets from different sources. How to integrate datasets with different attributes and show reviews of a restaurant from different source is the idea of this project.
